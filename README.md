# ❤️ 心形弹窗动画

一个浪漫的心形弹窗动画程序，运行时会在屏幕上呈现一颗由温馨语录组成的心形图案，然后散布填满整个屏幕。

## ✨ 功能特性

- 💖 **心形动画**：弹窗依次出现组成心形图案
- 🎨 **多彩配色**：柔和明亮的背景色随机分配
- 💬 **温馨语录**：52条关心语录随机展示
- ⚡ **全屏散布**：心形展示后弹窗会散布填满整个屏幕
- 🎯 **快捷关闭**：按空格键或 ESC 键关闭所有弹窗

## 🚀 使用方法

### 方式一：直接运行 EXE（推荐）

1. 下载 `heart_popup.exe` 文件
2. 双击运行即可看到心形弹窗动画
3. 按 **空格键** 或 **ESC 键** 关闭所有弹窗

### 方式二：Python 源码运行

```bash
python heart_popup.py
```

### 方式三：浏览器运行

打开 `heart_popup.html` 文件在浏览器中运行，点击屏幕任意位置关闭弹窗。
